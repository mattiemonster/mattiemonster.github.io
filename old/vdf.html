VDF

VDF - Vortex Data File - is a small, lightweight file format that can store basic types (int, string, float, etc) via binary serialisation and a friendly API. It's written in C# and can be used when referencing a DLL. There is also an explorer program (not fully functional). The vdf files (.vdf) are usually quite small, with only around 800 bytes of obligatory data.

Creating a VDF

VDF vdf = new VDF(vdf_name)

Item Types

Boolean - VDFBoolItem

Double - VDFDoubleItem

Float - VDFFloatItem

Int - VDFIntItem

Long - VDFLongItem

String - VDFStringItem

Base class - VDFItem

Creating an Item

VDFItem item = new VDFItem(name, initialValue);

Replace VDFItem with the class name of the type of item you'd like to store (See Item Types)

Creating a Catagory

VDFCatagory catagory = new VDFCatagory(catagoryName);

Adding an Item to a Catagory

catagory.AddItem(item);

Removing an Item from a Catagory

catagory.RemoveItem(item);

Add or Remove an Item from the root

vdf.AddItem(item); or vdf.Remove(item);

Load a VDF

VDFReader.LoadVDF(path); (RETURNS VDF OBJECT)

VDFReader.LoadVDF(streamWithVDFFile); (RETURNS VDF OBJECT)

Save a VDF

vdf.Save();

This method saves the VDF at the path stored at the VDFs 'savePath' value

vdf.Save(stream);

This method saves the VDF via a provided stream. The stream must already have a path set and be initialised.

vdf.Save(path);

This method saves the VDF by creating a new stream at the path provided and saves via that stream.

There is a .toString() override for a VDF, it prints out all items, catagories and items within catagories and prints out their names and values.
